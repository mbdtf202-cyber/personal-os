# Personal OS 🚀

一个现代化的个人管理系统，集成健康追踪、学习管理、项目管理、交易记录等多个模块，帮助你更好地管理数字生活。

## ✨ 核心特性

### 🎨 精致的 UI 设计
- 玻璃态设计风格，现代艺术感
- 流畅的动画和过渡效果
- **6 种精美主题**：极简白、柔和蓝、清新绿、优雅紫、温暖米、深邃黑
- 完整的主题系统，所有组件统一风格

- 内存缓存层，提升数据访问速度
- 数据库查询优化和批量加载
- 性能监控和指标追踪
- 离线支持和 PWA 功能

### 🤖 智能工作流
- **自定义工作流**：创建个性化的自动化流程
- 手动触发或定时执行
- 工作流步骤管理

- 智能数据分析和洞察

### 📊 数据可视化

## 🛠️ 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **数据库**: SQLite + Prisma ORM
- **样式**: Tailwind CSS 4
- **UI 组件**: Radix UI
- **图表**: Recharts
- **状态管理**: Zustand
- **数据请求**: TanStack Query
- **表单验证**: Zod

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 初始化数据库

```bash
npm run db:push
npm run db:seed
```

### 启动开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)

### 其他命令

```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start

# 数据库管理
npm run db:studio

# 运行测试
npm test

# 抓取新闻
npm run fetch-news
```

## 📦 项目结构

```
personal-os/
├── app/                    # Next.js App Router
│   ├── (dashboard)/       # 仪表板布局
│   ├── api/               # API 路由
│   └── globals.css        # 全局样式
├── components/            # React 组件
│   ├── ui/               # 基础 UI 组件
│   ├── layout/           # 布局组件
│   ├── dashboard/        # 仪表板组件
│   ├── analytics/        # 分析组件
│   ├── charts/           # 图表组件
│   ├── ai/               # AI 功能组件
│   ├── workflows/        # 工作流组件
│   └── notifications/    # 通知组件
├── lib/                   # 工具库
│   ├── services/         # 业务服务
│   ├── themes/           # 主题系统
│   ├── cache.ts          # 缓存系统
│   ├── performance.ts    # 性能监控
│   ├── offline-storage.ts # 离线存储
│   └── realtime-sync.ts  # 实时同步
├── hooks/                 # React Hooks
├── prisma/               # 数据库
│   ├── schema.prisma     # 数据模型
│   └── seed.ts           # 种子数据
└── public/               # 静态资源
```
